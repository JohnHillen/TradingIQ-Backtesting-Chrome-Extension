If a field has not been filled, it will not be changed in the Strategy settings.

If enabled the Min ATR will set as pairs, this will reduce the backtest cycles, eg.:
Min ATR Profit: 0.2,0.5
Min ATR Stop: 1,2
enabled: (Profit: 0.2, Stop: 1), (Profit: 0.5, Stop: 2) = 2 cycle
disabled: (Profit: 0.2, Stop: 1), (Profit: 0.2, Stop: 2), (Profit: 0.5, Stop: 1), (Profit: 0.5, Stop: 2) = 4 cycles
